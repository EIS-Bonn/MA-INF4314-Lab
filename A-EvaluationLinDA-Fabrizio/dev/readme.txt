steps to install and launch LinDa and virtuoso

Open the terminal in Ubuntu and enter ther following commands

1. ps aux | grep virtuoso

2. sudo kill <number of the process virtuso 6.1>


3. sudo /usr/local/virtuoso-opensource/bin/virtuoso-t +config /usr/local/virtuoso-opensource/var/lib/virtuoso/db/virtuoso.ini 


4. cd /home/comp/visualisation/backend

5. nodemon &

6. cd /home/comp/visualisation/frontend

7. grunt serve
